# Single-page-cv

# Objetivo
El objetivo es hacer una pagina simple del curriculum teniendo en cuenta la estructura semantica HTML para su facilidad de comprension y posiblemente modificable a futuro.
* Se utiliza HTML semantico
* Metaetiquetas SEO
* Etiquetas Open Graph (OG)
* Favicon
# Como abrir el proyecto
Abrí el archivo `index.html` en tu navegador, o visitá la URL del proyecto.
